# Community Tax and Management Services

A responsive client website focused on a delivering marketing content and implementing a secure contact form system. The website was developed with a logical approach to content strategy using the intial intent provided.

**Live Site:** https://communitytaxandmanagementservices.com/

## Project Overview and Strategy

This project involved building a professional website for a local tax service company. The development strategy focused on creating a single page narrative that effectively advertises the company and its services. Interested users would then be directed to the contact form that would be featured on its own page.

## Key Features

- Fully responsive layout
- Contact form with validation
- Secure form submission
- File upload support
- Spam protection
- Environmental variable protection for API keys
- Deployed optimized production build

## Tech Stack

- **Frontend:** Typescript, Tailwindcss, React + Vite, SweetAlert, GSAP
- **Form Handling:** Web3Form, hCaptcha
- **Deployment:** Netlify
